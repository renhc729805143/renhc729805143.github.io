课程笔记

函数一等公民

  1. 函数 有双重身份，即可以当做函数来调用，可以当做对象来访问属性或调用方法

    • 通过使用的方式来确定身份
    • 如果直接在函数名后加 一对圆括号，此时就是函数身份
    • 如果在函数名后有. | [] ，此时身份为对象，即函数访问属性或调用方法

  2. 函数可以作为另一个函数的实参

    • 回调函数

  3. 函数可以嵌套使用

  4. 函数 可以作为另一个函数返回值

    • 闭包

  5. 函数可以限定作用域

函数相关属性

  1. arguments

    • 伪数组对象
    • 已数组形式 来存储函数在执行时所传入的所有实参
    • callee 返回正在被执行函数; 匿名函数的递归调用
    • length 实参个数

  2. caller: 返回调用函数的 函数

  3. length: 定义形参的个数

  4. name: 存储函数的名字
